# EOS TEST

## OBJECTIVE

Pull most recent 10 blocks from eos chain.



### INSTALLATION



```bash


# install all node modules
npm install;

# start node server
# http://localhost:3000/
npm start;



# http://localhost:7777/
# 
npm run express


```


The following url's should now be accessible.

##### front end
http://localhost:3000

##### express server
http://localhost:7777
